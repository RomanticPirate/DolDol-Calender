요소별 상세

요소동작결과
[✕] 닫기클릭모달 닫힘
[😀] 이모지클릭팔레트 토글 (8탭). 바깥 클릭 시 닫힘. 선택한 이모지 → 제목 커서 위치에 삽입
🏷️ 카테고리클릭설정에서 만든 카테고리만 표시 (아이콘+이름). 라디오 선택. 여기서 추가/수정 불가
📅 종료일선택디폴트 = 시작일
🕐 종일토글디폴트 OFF. ON→시간 숨김, OFF→시간 표시
👥 일정 참가자체크박스디폴트: 전체 + 모든 멤버 체크. "전체" 토글→하위 동기화. 하위 해제→"전체" 자동 해제
🔔 알림 ⓘ체크박스디폴트: 1일 전 + 1시간 전. 다중 선택. ⓘ 클릭→"일정 참가자에게만 알림이 갑니다" 툴팁 (바깥 클릭 시 닫힘)
사용자 설정체크→펼침주/일/시/분 입력 + [+]로 추가. 추가된 항목 [✕]로 삭제
📝 메모입력줄바꿈 지원 (pre-line)
[저장]클릭1회 클릭으로 즉시 저장. 제목 필수. 폰트 +3pt, 가로 넓게
[취소]클릭모달 닫힘. 폰트 +3pt, 텍스트 크기만큼 작게
스크롤 위치모달 열릴 때 항상 맨 위로 스크롤 초기화
