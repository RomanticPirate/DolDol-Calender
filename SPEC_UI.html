4.4 달력 내 일정 미리보기 (핵심 변경)

네이버 캘린더 스타일 날짜 셀의 세로 높이를 충분히 확보하여, 일정 제목이 색상 라벨 위에 텍스트로 보이게 표시합니다. 이 미리보기 UI는 향후 홈 화면 위젯에서도 동일하게 사용됩니다.
항목설명
표시 형태색상 라벨 배경 위에 일정 제목 텍스트. 라운드 모서리.
텍스트 색상 자동 조절배경색이 어두우면 → 흰색 텍스트 / 배경색이 밝으면 → 어두운 텍스트
긴 텍스트일부만 보이고 뒤는 잘림 (overflow hidden, 말줄임 없음)
최대 표시 수셀 높이에 따라 2~3개. 넘치면 "+N" 표시
다일간 일정여러 날에 걸쳐 바 형태로 연결 표시

📅 날짜날짜 선택필수값. 디폴트: 현재 선택된 날짜
📅 종료일날짜 선택디폴트: 시작일과 동일
🕐 종일 일정 토글클릭디폴트: OFF. ON → 시간 필드 숨김, OFF → 시간 필드 표시

🔄 반복드롭다운반복 없음 / 매일 / 매주 / 매월 / 매년

👤 알림 대상드롭다운맨 위: "모두 알림" (전체 멤버) / 그 아래: 개별 멤버 이름 목록
🔔 알림 체크박스체크/해제다중 선택 가능. 디폴트: "1일 전" + "1시간 전" 선택됨
☐ 사용자 설정체크커스텀 알림 입력 영역 펼침. 주/일/시/분 숫자 입력 + [+] 버튼으로 추가. 추가된 항목은 삭제/수정 가능
[+] 커스텀 알림 추가클릭입력한 주/일/시/분 값으로 알림 항목 추가. 추가된 항목 옆에 [✕] 삭제 버튼

[취소] 버튼클릭모달 닫힘 (✕와 동일). 텍스트 크기만큼 작게. 폰트 기본+3pt.
삭제된 항목 📁 카테고리 선택 — 삭제됨. 색상 라벨로 구분 충분.
