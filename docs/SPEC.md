# 돌돌 캘린더 — 공유 달력 앱

## 1. 앱 개요

**앱 이름**: 돌돌 캘린더
**컨셉**: 가족/커플이 기념일과 일정을 함께 기록하고 공유하는 달력 앱
**레퍼런스**: TimeTree, 네이버 캘린더
**저작권**: © 2026 JJW. All rights reserved.

## 2. 핵심 기능 요약

### 2.1 캘린더
- 월간 달력 뷰 (메인 화면, 항상 표시)
- 고정 높이 셀 (95px), 일정 유무와 무관하게 균일한 레이아웃
- 셀 내 일정 미리보기 (카테고리 아이콘 + 색상 라벨 + 제목, 최대 4개 + "+N")
- 색상 밝기 자동 판별 (어두운 배경→흰 글씨, 밝은 배경→검정 글씨)
- 선택일: 셀 전체에 보라색 테두리 + 연보라 배경
- 한국 공휴일 자동 표시 (공공데이터 API + 내장 데이터 fallback)
- 연도→월→일 3단계 날짜 빠른 이동 팝업
- 좌우 스와이프 월 이동

### 2.2 일정 관리
- 일정 생성/수정/삭제 (모달 방식)
- 종일 일정 / 시간 일정 (디폴트: 종일 OFF, 시간 표시)
- 종료일 디폴트 = 시작일
- 반복 설정 (매일/매주/매월/매년)
- 카테고리 시스템 (아이콘 이모지 + 색상, 설정에서 관리)
- 이모지 팔레트 (8탭, Unicode 표준, 직접 구현)
- 댓글 기능 (일정별)
- D-Day 카운트 (매년 반복 일정에 자동 표시)
- 일정 패널에서 메모 미리보기 (2줄까지)
- 메모 줄바꿈 지원

### 2.3 일정 참가자
- 체크박스 다중 선택 ("전체" + 개별 멤버)
- "전체" 토글 시 하위 전원 동기화
- 하위 1개라도 해제 시 "전체" 자동 해제
- 미리보기/상세에서 참가자 이름 표시 (작성자 대신)

### 2.4 알림
- 다중 알림 시간 설정 (체크박스 방식)
- 프리셋: 시작 시간 / 5분 전 / 10분 전 / 30분 전 / 1시간 전 / 1일 전 / 1주 전
- 사용자 정의 알림 (주/일/시/분 직접 입력, 복수 추가/삭제 가능)
- 디폴트: 1일 전 + 1시간 전
- 알림은 일정 참가자에게만 발송 (인포 툴팁으로 안내)
- 프로덕션: Android AlarmManager (로컬) + 백그라운드 동기화

### 2.5 공유
- 1개 캘린더 공용 사용 (추가/변경 불가)
- 초대 코드(6자리)로 멤버 참여
- 멤버별 고유 색상 (댓글 아바타, 설정 미리보기)
- "내 일정만 보기" 토글 (로컬 저장, 앱 재시작 시 유지)

### 2.6 상단 타이틀 바
- 앱 이름 "돌돌 캘린더"
- 🔔TEST: 알람 테스트 버튼 (배포 시 제거)
- 👤 내 일정 토글 스위치
- 📜 히스토리 팝업 (최대 30개, 오래된 것부터 자동 삭제)
- ⚙️ 설정 팝업
- 하단 네비게이션 없음 (모든 서브 화면은 팝업/모달)

### 2.7 설정
- 프로필: 닉네임 수정 (변경 시 모든 댓글에도 반영), 내 색상 (팝업 선택, 첫 글자 미리보기)
- 캘린더: 캘린더 정보, 멤버 목록, 초대 코드 복사
- 카테고리 관리: 추가/삭제, 아이콘(이모지 13탭 260종) + 색상(감성 톤 15종)
- 데이터: .ics 익스포트 (iCalendar RFC 5545)
- 공휴일 API: 키 설정, 캐시 초기화

### 2.8 데이터 안전
- 쓰기 전 백업 (dd_backup)
- 삭제 시 전체 스냅샷 보존 (dd_delete_log)
- 동기화 충돌 시 양쪽 데이터 보존

### 2.9 온보딩
- 타이틀 이미지 (날짜별 자동 변경: ~2034→img1, 2034~2046→img2, 2046~→img3)
- "돌돌 캘린더 / 우리가 함께 하는 날들"
- 닉네임 입력 → 시작 → 기본 캘린더 + 샘플 카테고리 3개 + 샘플 일정 2개 자동 생성

## 3. 기술 스택

| 구분 | 데모 | 프로덕션 |
|------|------|----------|
| 프론트엔드 | HTML + CSS + Vanilla JS | 동일 (WebView) |
| 데이터 저장 | localStorage | Firebase Firestore |
| 동기화 | — | Firebase Realtime + WorkManager 백그라운드 |
| 공휴일 | data.go.kr API + 내장 | 동일 |
| 알림 | Web Notification (테스트) | Android AlarmManager (로컬) |
| 익스포트 | .ics 파일 다운로드 | 동일 |
| 배포 | 로컬 HTML | Android WebView APK |
