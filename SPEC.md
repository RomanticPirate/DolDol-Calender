# 돌돌 캘린더 — 공유 달력 앱

## 1. 앱 개요

**앱 이름**: 돌돌 캘린더
**컨셉**: 가족/커플이 기념일과 일정을 함께 기록하고 공유하는 달력 앱
**레퍼런스**: TimeTree, 네이버 캘린더
**저작권**: © 2026 JJW. All rights reserved.

## 2. 핵심 기능 요약

### 2.1 캘린더
- 월간 달력 뷰 (메인 화면)
- 네이버 캘린더 스타일 일정 미리보기 (색상 라벨 + 제목)
- 한국 공휴일 자동 표시 (공공데이터 API 연동 + 내장 데이터 fallback)
- 연도/월/일 빠른 이동 팝업
- 좌우 스와이프 월 이동

### 2.2 일정 관리
- 일정 생성/수정/삭제
- 종일 이벤트 / 시간 이벤트
- 반복 설정 (매일/매주/매월/매년)
- 색상 라벨 (10색, 밝기에 따라 텍스트 색상 자동 조절)
- 이모지 팔레트 (직접 구현, Unicode 표준)
- 댓글 기능 (일정별)
- D-Day 카운트

### 2.3 알림
- 다중 알림 시간 설정 (체크박스 방식)
- 기본 선택지: 시작 시간, 5분 전, 10분 전, 30분 전, 1시간 전, 1일 전, 1주 전
- 사용자 정의 알림 (주/일/시/분 직접 입력, 복수 추가 가능)
- 알림 대상 선택 (전체 / 개별 멤버)
- 디폴트: 1일 전 + 1시간 전

### 2.4 공유
- 1개 캘린더를 공용으로 사용 (캘린더 추가/변경 불가)
- 초대 코드(6자리)로 멤버 참여
- 멤버별 고유 색상으로 생성자 구분
- Firebase 기반 실시간 동기화 (프로덕션)

### 2.5 데이터 안전
- 데이터 유실 방지 최우선 설계
- 쓰기 전 백업 → 쓰기 → 검증 패턴
- 동기화 충돌 시 양쪽 데이터 보존 (덮어쓰기 금지)
- 삭제 기록 관리 (실수 삭제 복구 가능)

### 2.6 익스포트
- iCalendar (.ics) 표준 형식 (RFC 5545)
- Google 캘린더, Apple 캘린더, Outlook 등 호환

## 3. 기술 스택

| 구분 | 데모 | 프로덕션 |
|------|------|----------|
| 프론트엔드 | HTML + CSS + Vanilla JS | 동일 (WebView) |
| 데이터 저장 | localStorage | Firebase Firestore |
| 동기화 | — | Firebase Realtime Sync |
| 공휴일 | data.go.kr API + 내장 | 동일 |
| 푸시 알림 | UI만 구현 | FCM |
| 익스포트 | .ics 파일 다운로드 | 동일 |
| 배포 | 로컬 HTML | Android WebView APK |
